# macOS framebuffer goldens

These screenshots are generated by the pinned macOS SDL emulator at 1200×1600.
Regenerate them with `ZEN_UI_RUN_EMULATOR=1 ZEN_UI_UPDATE_GOLDENS=1 ./spec/run smoke`.
They are review artifacts: Cocoa rendering and cover extraction are not stable enough
for exact comparison unless `ZEN_UI_GOLDEN_DIR` explicitly opts into it.
